Dotfiles
========

This is my collection of [configuration files](http://dotfiles.github.io/).

Usage
-----

Pull the repo along with all it's submodules

```bash
git clone --recursive git@github.com:alexpearce/dotfiles.git $HOME/.dotfiles
```

Deploy!

```bash
cd ~/.dotfiles
rake install
```
